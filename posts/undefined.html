  1. 新建任意一个文件夹，存放我们要写的diamante

  2. 新建一个java文件

    • 文件后缀名为.java
    • Hello.java
    • 【注意点】系统可能没有显示文件后缀名，我们需要手动打开

  3. 编写代码

    public class Hello{
    public static void main(String[] args) {
        System.out.print("Hello,Word!");
    }
}

  4. 打开cmd 运行javac java 文件，会生成一个class文件

  5. 运行java Hello
